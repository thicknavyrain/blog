Design notes

Plain and functional, with a simple dark mode

The layout uses modular CSS grid tiles so you can add, remove, or reorder cards without adjusting the layout. Dark mode is a single class toggle that respects prefers-color-scheme and keeps the palette restrained.

Built with plain HTML, CSS, and a hint of vanilla JavaScript.
